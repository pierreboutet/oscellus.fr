Une bière sur la base d'une Irish Red Ale, avec du houblon fuggles.


### la recette : 


#### Malts : 
* Pale Ale 8EBC ( 4,4Kg )
* Caramunich III 150EBC ( 200g )
* Carared 50EBC ( 300g )
* Chocowheat 1000EBC ( 50g )
* Spécial W 300EBC ( 200g )
* Fumé 8EBC ( 200g )

#### Houblons:
* Fuggles ( 55g + 45g )

#### Levure
Fermentis S-04
